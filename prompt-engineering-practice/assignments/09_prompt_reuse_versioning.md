# Assignment 09 - Prompt Reuse and Versioning

## Practical Task

Write two versions of a prompt for the same task, such as business ideas, a workout plan, or an email reply.

## Rules

- Write `Prompt v1` and `Prompt v2`
- Keep the same task in both versions
- Make version 2 clearer and more detailed

## What Will Be Checked

- both prompts are about the same goal
- version 2 shows a meaningful improvement
- the changes are practical and easy to understand
