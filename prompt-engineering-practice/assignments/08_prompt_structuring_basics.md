# Assignment 08 - Prompt Structuring Basics

## Practical Task

Write one structured prompt on a real topic such as simple business ideas, email writing, or a study plan.

## Rules

- Include these sections: Role, Task, Context, Constraints, Output Format
- Keep the topic beginner-friendly
- Ask for a clearly formatted answer

## What Will Be Checked

- correct prompt structure
- clarity of each section
- useful output format
