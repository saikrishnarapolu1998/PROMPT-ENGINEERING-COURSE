# Assignment 04 - Multi-Shot Prompting

## Practical Task

Write a prompt with multiple examples that teaches the model how to turn simple product names into short marketing lines.

## Rules

- Include at least 4 examples
- Keep each marketing line short
- Ask the model to generate 1 new line at the end

## What Will Be Checked

- enough examples are provided
- examples follow one clear pattern
- generated output matches that pattern
