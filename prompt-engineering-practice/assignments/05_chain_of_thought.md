# Assignment 05 - Chain-of-Thought Prompting

## Practical Task

Write a prompt that asks the model to solve a simple logic or math problem step by step before giving the final answer.

## Rules

- Use a short problem
- Ask for step-by-step reasoning
- End with a final answer line

## What Will Be Checked

- reasoning steps are requested clearly
- problem is simple and realistic
- final answer is easy to find
