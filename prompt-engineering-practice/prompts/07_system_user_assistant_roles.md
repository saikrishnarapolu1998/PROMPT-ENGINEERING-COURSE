# 07 System, User, and Assistant Roles

## What it means

This approach separates the prompt into roles. The system sets behavior, the user gives the task, and the assistant returns the response.

## When to use it

Use it when you want more control over tone, rules, or behavior.

## Small example

The system says, "You are a beginner-friendly tutor." The user says, "Explain photosynthesis simply." Then the model answers in that style.

## Key takeaway

Separating roles makes prompts cleaner and easier to control.
