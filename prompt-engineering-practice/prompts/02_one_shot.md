# 02 One-Shot Prompting

## What it means

One-shot prompting means you give the model one example before asking it to do the real task.

## When to use it

Use it when the model needs a quick pattern to follow.

## Small example

You show one example of a formal email, then ask the model to write a new formal email on another topic.

## Example prompt

Example:
Input: Write a polite leave request for one day.
Output: Dear Manager, I would like to request one day of leave for personal reasons.

Now write a polite leave request for two days.

## Sample output

Dear Manager, I would like to request two days of leave for personal reasons. I will ensure my work is up to date before my absence.

## Key takeaway

One good example can improve output quality a lot.
