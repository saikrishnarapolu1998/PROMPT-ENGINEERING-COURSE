# 10 Output Formatting

## What it means

Output formatting means telling the model exactly how the answer should look.

## When to use it

Use it when you want clean results that are easy to read, copy, or process.

## Small example

You ask the model to return the answer as a table with columns for idea, cost, and difficulty.

## Example prompt

Suggest 3 small business ideas for a student. Return the answer as a table with columns: Idea, Cost, Difficulty.

## Sample output

| Idea | Cost | Difficulty |
| --- | --- | --- |
| Notes selling | Low | Easy |
| Resume help | Low | Easy |
| Snack delivery | Medium | Medium |

## Key takeaway

If you want structured output, ask for it directly.
