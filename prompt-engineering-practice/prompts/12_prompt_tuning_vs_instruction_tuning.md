# 12 Prompt Tuning vs Instruction Tuning

## What it means

Prompt tuning changes how a model is guided for a task, while instruction tuning changes the model itself through training on instruction-following examples.

## When to use it

Use normal prompting when you are working as a user. Learn this topic to understand the difference between writing better prompts and changing the model during training.

## Small example

If you rewrite your prompt to get better answers, that is prompt work. If a company trains the model to follow instructions better across many tasks, that is instruction tuning.

## Example prompt

Question:
What is the difference between prompt tuning and instruction tuning?

## Sample output

Prompt tuning improves how you guide a model for a specific task through prompt design. Instruction tuning improves the model itself through training so it follows instructions better across many tasks.

## Key takeaway

Prompting changes the input. Instruction tuning changes the model.
