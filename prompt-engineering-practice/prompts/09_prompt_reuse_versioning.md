# 09 — Prompt Reuse & Versioning

## What it means

Prompt reuse means using the same prompt structure again for similar tasks.

Prompt versioning means improving a prompt over time and saving the improved versions clearly.

## Why this matters

In real projects, prompts are not written only once.
They are often:
- reused
- refined
- compared
- improved

## Simple example

- Prompt v1 = basic
- Prompt v2 = clearer and more structured
- Prompt v3 = even more controlled

## Reuse example

A business-ideas prompt can be reused by changing:
- audience
- budget
- time available
- business goal

## Example prompt

Prompt v1:
Suggest business ideas.

Prompt v2:
Suggest 5 beginner-friendly business ideas for a college student with a low budget.

## Sample output

Version 2 is likely to give more relevant answers because it defines the user type, skill level, and budget clearly.

## Key idea

Good prompts are often built in versions, not in one attempt.
