# 06 Zero-Shot Chain-of-Thought

## What it means

Zero-shot chain-of-thought means you ask for step-by-step reasoning without giving any examples.

## When to use it

Use it when you want better reasoning but do not want to build a long prompt with examples.

## Small example

You write, "Solve this problem step by step," and the model explains its reasoning before the answer.

## Example prompt

Solve this step by step:
If a shop sells 4 notebooks for $3 each, how much money does it make in total?

## Sample output

Each notebook costs $3.
There are 4 notebooks.
Multiply 4 by 3.
Total money made = $12.

## Key takeaway

A short reasoning instruction can sometimes improve results immediately.
