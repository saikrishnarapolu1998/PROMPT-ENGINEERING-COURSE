# 05 Chain-of-Thought Prompting

## What it means

Chain-of-thought prompting asks the model to think through a problem step by step before giving the final answer.

## When to use it

Use it for reasoning tasks like math, logic, planning, or decision-making.

## Small example

Instead of asking only for the answer to a word problem, you ask the model to solve it step by step.

## Key takeaway

Step-by-step reasoning can improve difficult answers.
