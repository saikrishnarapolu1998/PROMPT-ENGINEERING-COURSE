# One-Shot Prompting

## What It Means

One-shot prompting means giving the model exactly one example before asking it to perform the task.

## In This Example

The prompt shows one example of changing a direct sentence into a professional business tone, then asks the model to do the same for a new input.

## Why It Works

- The model sees the expected transformation.
- The response style is demonstrated once.
- The output format is easy to follow.

## Example Prompt

```text
Example:
Input: Why isn't this done yet?
Output: Could you provide a quick status update on this task?
```

## Use Cases

- Tone conversion
- Text rewriting
- Format conversion
- Short response generation
