# Zero-Shot Prompting

## What It Means

Zero-shot prompting means asking the model to do a task without giving any example outputs.

## In This Example

The prompt asks the model to rewrite a paragraph about AI into 5 simple bullet points for a beginner.

## Why It Works

- The task is clearly defined.
- The expected format is mentioned.
- The audience is specified as a beginner.

## Example Prompt

```text
Rewrite the following paragraph in 5 simple bullet points for a beginner.
```

## Use Cases

- Summarization
- Rewriting
- Basic classification
- Simple content generation
